# hw2 — Car Factory

Консольное приложение на C#, демонстрирующее паттерн «Фабрика» для создания объектов автомобилей с выводом их описания.

## Структура проекта

```
hw2/
├── Program.cs              — точка входа, цикл ввода
├── CarType.cs              — enum с поддерживаемыми марками
├── Interfaces.cs           — интерфейсы ICar, IElectric, IMechanicalEngine, IManual, IAutomatic
├── ACar.cs                 — абстрактный базовый класс
├── CarFactory.cs           — фабрика, возвращающая ICar по CarType
└── Cars/
    ├── AElectricCar.cs         — абстрактный электромобиль
    ├── AGasCar.cs              — абстрактный бензиновый автомобиль
    ├── AAutomaticElectricCar.cs — электромобиль с автоматом
    ├── AAutomaticGasCar.cs     — бензиновый автомобиль с автоматом
    ├── AManualGasCar.cs        — бензиновый автомобиль с механикой
    ├── TeslaCar.cs             — Tesla Model S
    ├── BmwCar.cs               — BMW 5 Series
    ├── ToyotaCar.cs            — Toyota Camry
    ├── FordCar.cs              — Ford Focus
    └── LadaCar.cs              — Lada Vesta
```

## Требования

- [.NET SDK](https://dotnet.microsoft.com/download) версии **8.0** или выше

Проверить наличие .NET:

```bash
dotnet --version
```

## Запуск

### Способ 1 — через `dotnet run`

```bash
cd hw2
dotnet run
```

### Способ 2 — сборка и запуск исполняемого файла

```bash
cd hw2
dotnet build -c Release
./bin/Release/net10.0/hw2
```
